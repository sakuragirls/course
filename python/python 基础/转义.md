## 1.python的转义符

1. \n:换行

2. \t:制表符：代表四个空格的位置

3. \r:覆盖

4. \b:删除

5. \\\\:两个反斜线表示一个\

#3.\t
print('hello\thello')#\t使两个hello之间相隔四个空格
#结果
"""
hello   hello
"""

#4.\r
print("我\r爱你")
"""
爱你
"""
#5.\b
print("helll\bo")#将\b的前一个字符删除
"""
hello
"""

#6.\
print('\\')
#结果
"""
\
"""

\n
print('我有\n一个最好的朋友，他叫侯默')
#结果
"""
我有
一个最好的朋友，他叫侯默
